# Urban Exposome Crimea

Интерактивная карта скрининговой экологической оценки по Крыму. Пользователь кликает точку на карте и получает значение выбранного фактора (свет, тепло, NO2, шум), нормативный ориентир и текстовую интерпретацию.

## 1) Что делает проект

- Показывает тематические слои экспосома: `light`, `heat`, `no2`, `noise`.
- Возвращает значение в выбранной точке (из GeoTIFF) и скрининговую интерпретацию.
- Поддерживает быстрый детерминированный режим (`demo mode`) без обязательной зависимости от внешних API.
- Поддерживает опциональную LLM-функцию как отдельный шаг: «Расширенная интерпретация (LLM)».

## 2) Pipeline: GEE → GeoTIFF → backend → frontend

1. **GEE (`gee/*.js`)**
- Формируются слои из спутниковых/геоданных коллекций.
- Выполняется агрегация (`median`) и сглаживание (`focal_mean`).
- Экспорт в GeoTIFF (`scale: 250`, CRS: WGS84/EPSG:4326).

2. **GeoTIFF (`data/*.tif`)**
- Backend читает растр и получает значение в точке (`lat/lon`) через `rasterio`.
- Для NO2 применяется перевод в оценку приземной концентрации (мг/м³).

3. **Backend (`backend/main.py`)**
- `GET /healthz`: статус сервиса, слоев, demo mode, LLM.
- `GET /ready`: readiness (200/503).
- `GET /api/analyze-single`: быстрый single-factor анализ (детерминированный).
- `GET /api/analyze-complex`: быстрый комплексный скрининг.
- `GET /api/analyze-single-advanced`, `/api/analyze-complex-advanced`: опциональная LLM-интерпретация.

4. **Frontend (`frontend/*`)**
- При открытии страницы сначала проверяет backend (`/healthz`).
- При холодном старте показывает статус: «Сервер анализа запускается, это может занять до 60 секунд».
- Показывает значение и базовую интерпретацию сразу после ответа fast endpoint.
- LLM выполняется отдельно по кнопке, не блокируя основной результат.

## 3) Как запустить локально

### Вариант A: быстрый запуск demo-режима (рекомендуется)

```bash
cd backend
cp .env.example .env
./run_demo.sh
```

Backend будет доступен на `http://localhost:8000`.

Для frontend можно поднять статический сервер из папки `frontend`:

```bash
cd frontend
python3 -m http.server 5500
```

После этого открыть `http://localhost:5500`.

### Вариант B: Docker

```bash
docker build -f backend/Dockerfile -t urban-exposome-backend .
docker run --rm -p 8000:8000 --env-file backend/.env.example urban-exposome-backend
```

## 4) Ограничения проекта

- Это **скрининговая экологическая оценка**, не медицинская диагностика и не клиническое заключение.
- Разрешение данных ~250 м/px: локальные микроситуации (двор/улица/этаж) могут сглаживаться.
- В `demo mode` названия локаций намеренно упрощены (координаты/сектор).
- Опциональная LLM-интерпретация зависит от внешнего API и может быть отключена.
- Для части слоев используются модельные/прокси-параметры (например, акустический слой), а не прямые полевые измерения в каждой точке.
